General Information:

  • The Standard Form of a Polynomial is: y = axn + bxn − 1 + cxn − 2 + ... + k.

Graphing:

  • The graph of a Polynomial can be easily drawn using a number line and looking at x-intercepts.
  • In order to find the x-intercepts, you need to factor the Polynomial, and there are 2 methods in in doing this:
    1. Using Synthetic Division (Quick & Easy Method)

        2. Given f(x) = 3x3 + 2x2 + 1

      1. First find all possible zeros (Find all factors of the constant (k))

        2. Example: ±1

      2. Plug in the factors into the Polynomial until you find an actual zero

        3. Example:
        f(1) = 3 + 2 + 1 ≠ 0
        f(−1) = −3 + 2 + 1 = 0

      3. Set up a diagram like this:
        4. zero coefficients
        ↓

        Example:

        −1 3 2 1
        ↓
        3
      4. Pull the first number down
    2. Using Polynomial Long Division (If All Else Fails - Longer, Tedious Method)

        3. Given f(x) = 3x3 + 2x2 + 1

      1. First find all possible zeros (Find all factors of the constant (k))

        2. Example: ±1

      2. Plug in the factors into the Polynomial until you find an actual zero

        3. Example:
        f(1) = 3 + 2 + 1 ≠ 0
        f(−1) = −3 + 2 + 1 = 0

      3. Set up a diagram like this:
        x+1 3x3 + 2x2 + 1
      4. Then solve the same way you would do to solve long division problems
        3x2 − x + 1
        x+1 3x3 + 2x2 + 0x + 1
        − (3x3 + 3x2)
        −x2 + 0x
        − (−x2 − x)
        x + 1
        − (x + 1)
        0
