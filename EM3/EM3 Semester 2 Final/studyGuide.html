Inverse Functions:

  • An inverse function is a function that reverses another function
  • To find the inverse function of any function, f(x) = x, set the x value to f-1(x) and the f(x) value to x and solve for f-1(x) to find the inverse function of that function
  • For example, in f(x) = 3x2 + 1
  • The inverse would be f-1(x) =
    x − 1
    3
  • The graph of a function is reflected over the the line y = x to produce the inverse function

Function Composition:

  • A function composition means that a function is inside a function
  • A function composition would look like this: f(g(x))
  • To solve a function composition, you would have to input the first function as an input for the second function
  • For example, when f(x) = 3x-2 and g(x) = x2
    g(f(x)) = (3x-2)2, which also simplifies to g(f(x)) = 9x2 - 12x + 4

    Vectors:

    General Information:

    • A vector has a direction and a magnitude. Uh! Uh! Uh!
    • A vectors component form is ⟨ Δx, Δy ⟩
    • The magnitude can be derived from component form by x2 + y2
    • || v|| can be used to represent the magnitude of a vector
    • A unit vector is a vector of magnitude 1 unit
    • i represents ⟨1, 0⟩
    • j represents ⟨0, 1⟩

    Combining Vectors:

    • To graphically add vectors d and v, draw d and then draw v starting at where d ended. Finally, draw the resulting vector (from the starting point of d to the ending point of v) as d + v
    • In order to add vectors, add the component forms
    • In order to subtract vectors, subtract the component forms
