  • These come in the forms:
    • r = a ±b θ Symmetry with x-axis 0 ≤ θ ≤ 2π
    r = a ± b θ Symmetry with y-axis
  • If a > b, then there is no loop, rather a dimple that is never on the origin.
  • If a < b, then there is an inner loop.
