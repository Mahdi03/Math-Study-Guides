    Methods Of Testing Convergence/Divergence:

    n-th Term Test:
  • This should be the first test used with most series unless you already see a convering Geometric Series.
  • Simply take the limit of the sequance, and if it approaches any other value than 0, the series will diverge.
  • When the limit of the sequence converges to 0, another test needs to be tried out.
    • Geometric Series Test (GST):
  • Use the GST when you can find a common ratio r. And recall from last year, that if |r| ≥ 1, the series will diverge, and if |r| < 1, the series will converge.
    • Telescoping Series:
  • These series have alternating terms that end up canceling out and leave you with only the first few terms as the value that the entire series converges to.
  • Usually these series will involve Partial Fraction Decomposition and then finding the first couple of terms until you feel confident that the rest will cancel out.
    • Integral Test:
  • This test can be tried when other methods do not work, it is quite lengthy as it requires the usage of improper integration from the last unit.
  • Simply put, when the improper integral of the sequence converges, the infinite series of the sequence converges, and if the improper integral of the sequence diverges, so does the series. However, if the integral converges to a certain value, this does not mean that the infinite series converges to the same value.
  • In order to use this test, the function must be decreasing, continuous, and positive when x ≥ 1, and this needs to be shown every time.
    • P-Series Test:
  • If you can simplify your series into the form of a p-series (as shown below) then you can use the simple rule to check for convergence or divergence.

  • Similar to the Direct Comparison Test used with Improper Integrals, come up with another function that you can find the limit of.
  • If the larger series converges, then the smaller series converges, and if the smaller series diverges, then the larger series diverges.

    Alternating Series Test (AST):
  • Use this test when you have (-1)n or (-1)n + c (where c is any number) within your sequence.
  • If this is the case, create a new an without the alternating negative. If you can show that an = 0 and you can show that the series of an continuously decreases starting from a point, then the original alternating series converges.
  • Use this test to only prove convergence, if you find that an ≠ 0, then it diverges by the nth term test.
  • In order to prove whether it conditionally converges or if it absolutely converges, after using the AST to prove that the alternating series converges, remove the alternating part and use another test to show that the series without the alternating part either converges or diverges. If the alternating series converges and the normal series diverges, then the series as a whole conditionally converges. And if both converge then the series as a whole absolutely converges.

    • Power Series:

  • A power series is an infinite series that can be rewritten as a polynomial of infinite degree.
  • You can have Power Series centered at a certain point a, written in general form as such:
    • cn(x − a)n c0 + c1(x − a)1 + c2(x − a)2 + c3(x − a)3 + ... + cn(x − a)n + ...
  • Power series converge to a function, in terms of just x, and where it converges to the function actually depends. It could converge in select amount of value defined by the center and the radius of the Interval of Convergence (IOC), it could converge at just the center, or it could converge for all real numbers.
  • As for the endpoints in the IOC, they need to be tested individually in order to see if they are included in the IOC.
  • If you see that the series is geometric, then you can skip having to use the Ratio Test to solve for what the series converges to and finding the IOC as well as the radius.
  • Finding the IOC can be done by using the GST (if applicable) or the Ratio Test.
  • Using the GST, you identify a Geometric Series and find a common ratio r, setting that −1 ≤ r ≤ 1 and simplifying to find the IOC.
  • Using the Ratio Test, you have the absolute value of the function ≤ 1, which means that the function is ≤ 1 and ≥ −1
