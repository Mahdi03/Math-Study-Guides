Extreme Value Theorem or "EVT":
  • If a function f is continuous on the closed interval [a, b], then f must have an absolute minimum and maximum value at least once.
  • If a function f is continuous on the open interval [a, b], then f may have a local minimum and/or maximum value.
  • In order to find the minimum or maximum of a graph, take the derivative and solve for x when the derivative is either equal to 0 or undefined.
  • The reason this works is because when the function switches from increasing to decreasing and vice-versa, the function either curves, corners, or cusps. And the slope of the tangent line at the middle of these switches is either 0 or undefined.
  • Whenever the derivative is equal to 0 or undefined, those values for x are called critical points.
  • Once you have the critical points, plug them back into the original f, as well as the endpoints if the function is on a closed interval. Then rank the minimum and maximum of those values. If any of these values is also undefined, then it is not a critical point.
