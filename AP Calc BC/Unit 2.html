Theorems:

    Extreme Value Theorem or "EVT":
  • If a function f is continuous on the closed interval [a, b], then f must have an absolute minimum and maximum value at least once.
  • If a function f is continuous on the open interval [a, b], then f may have a local minimum and/or maximum value.
  • In order to find the minimum or maximum of a graph, take the derivative and solve for x when the derivative is either equal to 0 or undefined.
  • The reason this works is because when the function switches from increasing to decreasing and vice-versa, the function either curves, corners, or cusps. And the slope of the tangent line at the middle of these switches is either 0 or undefined.
  • Whenever the derivative is equal to 0 or undefined, those values for x are called critical points.
  • Once you have the critical points, plug them back into the original f, as well as the endpoints if the function is on a closed interval. Then rank the minimum and maximum of those values. If any of these values is also undefined, then it is not a critical point.
    • Mean Value Theorem or "MVT":
  • If the function g is continuous on the closed interval [a, b] and differentiable on the open interval (a, b), then there exists at least one x-value, x = c in (a, b) for which:

  • The process of finding an antiderivative is called antidifferentiating, and you must always assume a constant c.

  • Determine the sign of the derivative of f to the left and right of the critical values by creating a sign line and plugging in points (similar to graphing polynomials from EM3 Polynomial Functions)

  • If the derivative is positive, then the function is increasing.
  • If the derivative is negative, then the function is decreasing.
  • f(x) is concave up if f '(x) is increasing.
  • f(x) is concave down if f '(x) is decreasing.
  • If the second derivative is negative, then the function is concave down.
  • If the second derivative is positive, then the function is concave up.
  • If the function is concave up, then the tangent lines are underneath the graph.
  • If the graph is concave down, then the tangent lines go above of the graph.
    • Concavity Test:
    • Concave up where f ''(x) > 0
    • Concave down where f ''(x) < 0
    Point of Inflection:
  • A point where the graph of a function has a tangent line and where the concavity changes it a Point of Inflection.
  • A Point of Inflection occurs when the second derivative is equal to 0 or undefined, in other words, a critical point of the second derivative.
