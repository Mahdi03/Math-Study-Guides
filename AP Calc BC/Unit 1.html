The Sandwich (Squeeze) Theorem:

Suppose that g(x) ≤ f(x) ≤ h(x) for all values in some open interval containing c, except possibly at c itself. Suppose also that g(x) = h(x) = k. Then f(x) = k .

So basically come up with a g(x) and an h(x) that have the same limit in order to find the limit of f(x).

Continuity:

    f(x) is continuous at a point c of its domain if and only if f(x) = f(c) .

    3 Conditions:

    1. f(c) is defined
    2. f(x) exists
    3. f(x) = f(c)
  • If a function f is not continuous at x = c, then we say f is discontinuous at c and c is a point of discontinuity

    Intermediate Value Theorem or "IVT":

    If f(x) is continuous on [a, b] (a closed interval) and k is in [f(a), f(b)], then there must exist some c in [a, b] such that k = f(c).
