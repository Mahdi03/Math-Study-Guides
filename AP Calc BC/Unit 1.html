Limits:

  • Limits help define End Behavior of functions as well as the behavior of graphs at certain points.
    • x c f(x) = L
  • Read as: "The limit of f(x) as x approaches c is L"
  • The function values get closer and closer to L as the inputs get closer and closer to c.
  • In order for a limit to exist, its left-hand and right-hand limits must exist and be equal.
    • x c+/− f(x) = L
  • Read as: "The limit of f(x) as x approaches c from the right/left is L"

    • Limit Rules:

      Given:
      x c f(x) = L
      x c g(x) = K

    • x c b = b
    • x c x = c
    • x c f(x) ± g(x) = L ± K
    • x c f(x) × g(x) = L × K
    • x c b × f(x) = b × L
    • x c f(x)r/s = L r/s
    • x c f(x) ÷ g(x) = L ÷ K
    • x c f(g(x)) = f( x c g(x) ) = f(K)
    • x c ln(f(x)) = ln( x c f(x) ) = f(K)

The Sandwich (Squeeze) Theorem:

Suppose that g(x) ≤ f(x) ≤ h(x) for all values in some open interval containing c, except possibly at c itself. Suppose also that x c g(x) = x c h(x) .

So basically come up with a g(x) and an h(x) that have the same limit in order to find the limit of f(x).

Continuity:

  • f(x) is continuous at a point c of its domain if and only if x c f(x) = f(c)

    • 3 Conditions:

    1. f(c) is defined
    2. x c f(x) exists
    3. x c f(x) = f(c)
  • If a function f is not continuous at x = c, then we say f is discontinuous at c and c is a point of discontinuity

    Intermediate Value Theorem or "IVT":

    If f(x) is continuous on [a, b] (a closed interval) and k is in [f(a), f(b)], then there must exist some c in [a, b] such that k = f(c)
