6.1: Cauchy-Euler Equations:

  • So far we have only focused on differential equations with constant coefficients, but now it's time to focus on non-constant coefficients. Cauchy-Euler Equations are differential equations of the form:
    • anxn + an − 1xn − 1 + ... + a1x + a0y = 0
  • We can suspect the solutions to be of the form y = xn, and by plugging it in to the equation, we start to see that when the substituted equation has n distinct roots, the general solution is:
    • y(x) = c1xm1 + c2xm2 + ... + cnxmn
  • When we have repeating roots, we multiply subsequent terms by another x. The set of solutions for a root of multiplicity k is:
    • {x, x x, x[ x]2, ..., x[ x]k − 1}
  • When we have complex roots, we solve for the roots m, put them back into y = em, and then using Euler's Identity (eiθ = θ + i θ) and the Superposition Principle to make our answer real. When the root is m = a ± bi, the general solution will be:
    • y(x) = c1xa b x + c2xa b x
