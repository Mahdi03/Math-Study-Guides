7.1: LaPlace Transforms:

  • When we have differential equations in the form of y'' + y = f(t), we could have initial conditions and f(t) might not always be continuous. For this reason, we cannot use the annihilator since if it is not continuous, it would not be differentiable. If f(t) was piecewise, we would need to solve the differential equation for each piece.
  • We can try to "smooth out" these piecewise equations through, since if we integrate the piecewise, our result will be differentiable.
  • The Fundamental Theorem of Calculus connects the dots between differentiation and integration, and the properties of the integrand are that it must be piecewise continuous on the integrating domain.
  • Since we want our integral to converge, and to converge to some function of a different variable (notice how in the FTLI the function goes from f(t)dt to F(x) i.e. a change in variable) we want to add an exponential decay function to force infinite values to converge in most cases, so to this we add an extra independent variable that remains at the end of integration.
  • We call this method of integration the LaPlace Transform, and the LaPlace Transform of f(t) is given by the following equation as long as the integral converges:
    • f(x) = e−stf(t)

      7.2: Inverse LaPlace Transforms:

    2. The inverse of a LaPlace Transform is defined by F(s) = f(t) where f(t) = F(s) .
    3. There is a way to solve for the inverse LaPlace Transform but that involves complex numbers, for now we'll steer clear of that.
    4. Since LaPlace Transforms only exist for those functions where the LaPlace Transform integral converges, we need to make sure that the exponential decay in the integral is stronger than the actual input of the LaPlace Transform. Functions where an exponential function eventually beats the original function are said to be of exponential order. Functions that have vertical asymptotes on a positive t value (since our LaPlace Transform integral is integrated only over the positive t-axis) cannot be of exponential order.
    5. For all the LaPlace Transforms we've seen, we can deduce that if f(t) is piecewise continuous and contains no vertical asymptotes for t > 0:
      6. f(t) = 0
    6. By this idea, if the limit of the LaPlace Transform is not 0, the Inverse LaPlace Transform does not exist!!
