Conic Sections:

  • Recall the 4 conic sections introduced in EM2:
    • Parabolas:
  • Defined as:
    • The set of coplanar points equidistant from a point called the focus and a fixed line called the directrix.
    x2 = 4py y2 = −4px
    Circles:
  • Circles are in the 2-D realm (ℝ2) and so it's defined as:
    • The set of all coplanar points equidistant from one point
    (x − a)2 + (y − b)2 = r2
  • When you remove "coplanar" you open it up to ℝ3 where you get a sphere:
    • (x − a)2 + (y − b)2 + (z − c)2 = r2

    Parametric Functions:

  • Parametric functions define other variables which make up functions with one independent variable, called the parameter t.
  • We can take out the parameter and get a function with just the other variables, all defined by t by either
    1. Solving for t and substituting
    2. Using a trigonometric identity
    3. Using a "common" expression
  • The function x2 + y2 = 1 can be rewritten as x(t) = θ and y(t) = θ but only with the provided domain 0 ≤ θ ≤ 2π
