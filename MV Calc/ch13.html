13.1: Vector-Valued Functions:

  • The function notation for a vector-valued function is r: ℝ → ℝ3 (translates to a one-dimensional input and a three-dimensional output)
  • r(t) = ⟨x(t), y(t), z(t)⟩

13.2: The Calculus of Vector-Valued Functions:

  • Just as we apply calculus to regular functions, we can do the same to vector-valued functions by dealing with their individual components.
    • r(t) = ⟨x(t), y(t), z(t)⟩ r(t) = ⟨ x(t), y(t), z(t)⟩
    r'(t) = ⟨x'(t), y'(t), z'(t)⟩
    r(t) = ⟨ x(t), y(t), z(t)⟩
