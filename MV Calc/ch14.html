14.1: Introduction to Multivariable Functions:

  • Normally we reserve the vertical axis in a graph for the dependent variable.
  • Since z is the dependent variable, we want to see how z changes with respect to x and y. We only care about the traces in the xy-plane, we call these Level Curves.
  • When given a 3-D function, w = f(x, y), we draw 2-D Level Curves. When given a 4-D function w = f(x, y, z), we draw 3-D Level Surfaces.

14.2: Limits:

  • We can take 5 approaches to finding limits in 3-D space:
    1. Plug it in
    2. Algebra (factoring, conjugates)
    3. Squeeze (Sandwich) Theorem
    4. U-Substitution
    5. Polar equations and identities
  • We can suspect that the limit does not exist when the variables approach zero and the lowest degrees within the function match
  • When the above methods don't work, we can start to suspect the limit does not exist, in order to prove that the limit does not exist, we can substitute different values and functions for the variables (this simulates taking different paths to the same point) as long as the functions intersect with the point in question, and then if we get two different paths that lead to two different points, then we can conclude that the limit does not exist.
  • This method cannot prove if the limit exists as there are an infinite amount of paths to the given point.

14.4: Tangent Planes and Linear Approximation:

  • Using partial derivatives and vectors we can derive a tangent plane to the point on a function, given by the formula:
    • z − c = (x − a) + (y − b)
  • As it turns out, the formula above can also be used for the linearization of functions in ℝ3.
  • Another method of approximation involves the usage of differentials, given by the following for z = f (x, y):
    • dz = dx + dy
  • We can conclude a function is differentiable at a point when the partial derivatives exist and are continuous at the point in question. If the partial derivatives are not continuous, then the tangent plane approximation is bad.

  • Suppose z = f(x, y) is defined on D⊆ℝ2 with:
    1. f (x, y) is continuous on D
    2. D is a closed set (a region is considered a closed set if the set of all other points are considered open)
    3. D is a bounded set
    then f(x, y) achieves both an absolute maximum and a absolute minimum on D.
  • Remember to check all the components of the EVT before looking at critical points as well as boundaries and intersections of boundaries.

14.8: The Method of LaGrange Multipliers:

  • Another way we can find extrema of multivariable functions is by giving the function a constraint, which guarantees absolute extrema by the Extreme Value Theorem
  • Given a function z = f (x, y) constrained by g(x, y) = c, the extrema will satisfy (where λ is a constant):
    • ∇f (x, y) = λ∇g(x, y) g(x, y) = c
  • Using these, we can setup a system of multiple equations (break apart the gradient vectors into their equivalent components) and solve for our extrema.
