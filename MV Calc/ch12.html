12.2: Vectors:

  • Vectors are a value (a magnitude) and a direction (12 ft East).
  • Scalars are numbers that do not have a direction (12 lbs).
  • A vector's component form is ⟨Δx, Δy⟩
  • The magnitude can be derived from the component form by:
    • || v|| = x2 + y2 + z2
  • If two vectors have the same magnitude and direction, they are said to be equivalent vectors ( a = b)
  • A vector between two points A( x1, y1) and B( x2, y2) is said to be AB = ⟨x2 − x1, y2 − y1⟩
  • A unit vector is a vector of magnitude 1 unit, there are three "standard basis vectors" as follows:
    • i represents ⟨1, 0, 0⟩
    • j represents ⟨0, 1, 0⟩
    • k represents ⟨0, 0, 1⟩
  • To graphically add vectors a and b , draw a and then draw b starting at where a ended. Finally, draw the resulting vector (from the starting point of a to the ending point of b ) as a + b
  • In order to add vectors, add the component forms: A + B = ⟨x1, y1⟩ + ⟨x2, y2⟩ = ⟨x1 + x2, y1 + y2⟩
  • In order to subtract vectors, subtract the component forms: A − B = ⟨x1, y1⟩ − ⟨x2, y2⟩ = ⟨x1 − x2, y1 − y2⟩
  • Scalar multiplication means scaling a vector with a scalar number, so like c A = c⟨x, y, z⟩ = ⟨cx, cy, cz⟩
  • 0 A = 0
  • Vectors are parallel if they are scalar multiples of each other ( A = c B) .
  • A vector can be rewritten using the standard basis vectors x i + y j + z k Show Proof

  • Properties of the cross product:

    • || a ⨯ b|| = || a|| || b|| θ

12.5: Lines and Planes:

  • Describing a line in ℝ3 is different than in ℝ2
  • The vector equation of a line is given by:
    • r = r0 + t v Show Example
    Let's find the equation to the line that goes through the points A(2, 3, 6) and B(5, 6, 9):
    1. Imagine we start at the origin (0, 0, 0). First we need an inital vector r0 that will get us onto the line from the origin.
      2. r0 = ⟨2, 3, 6⟩
    2. Then we need to find a direction vector v = ⟨Δx, Δy, Δz⟩
      3. v = ⟨5 − 2, 6 − 3, 9 − 6⟩ = ⟨3, 3, 3⟩
    3. Finally putting it together, we get r = ⟨2, 3, 6⟩ + t⟨3, 3, 3⟩
  • The initial vector r0 gets you on the line, and v keeps you parallel to the line.
  • From the vector equation of a line you can derive the parametric equations of a line:
    • x = x0 + at
    y = y0 + bt
    z = z0 + ct
  • You also have the symmetric equations of a line, given by:

  • To check if two lines intersect we can simply let the equations equal each other and solve for any t values, if existent. Remember that just because the lines intersect, objects on those paths might not be at that point at the same time, therefore they might not collide.
  • Planes in space can also be depicted using vectors, but in a different method than that of a line.
  • Using the vector n normal to the plane is
    • n = ⟨A, B, C⟩
  • All points Q(x, y, z) such that PQ is orthogonal to n will result in a plane, so
    • n ⋅ PQ = 0 ⟨A, B, C⟩ ⋅ ⟨x − x0, y − y0, z − z0⟩ = 0
  • We can also depict planes in ℝ3 using the scalar equation of a plane:
    • A(x − x0) + B(y − y0) + C(z − z0) = 0
  • Or using the linear equation of the plane:
    • Ax + By + Cz = D
  • In order to find the equation of a plane containing three given points, choose one point as the center point and connect it to the two other points with 2 vectors, then you can take the cross product and find the normal.
    • w ⨯ v = n
  • Parallel planes have parallel normal vectors.

12.6: Cylinders and Quadratics:

  • A cylinder constitutes any planar curve that extends along an axis. Therefore, it would usually have an equation that is missing a variable. Earlier we saw an infinite parabola and an infinite circle, those are two examples of cylinders.
  • A surface of revolution is a surface formed by revolving a planar curve about an axis (so the cross-sections would correspond to the original planar curve).
